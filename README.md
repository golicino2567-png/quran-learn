# Учим Коран — прототип (Сура 2)

Сайт для заучивания аятов: арабский текст / транскрипция, перевод (Кулиев),
аудио чтеца (Мишари аль-Афаси) с режимом повтора.

## Как опубликовать (один раз, ~10 минут)

1. Зарегистрируйтесь на github.com (если нет аккаунта).
2. Создайте новый репозиторий (например `quran-learn`), загрузите туда
   все файлы из этой папки (через веб-интерфейс GitHub: "Add file" →
   "Upload files", перетащить всё содержимое папки).
3. Зарегистрируйтесь на vercel.com через GitHub-аккаунт.
4. На Vercel: "Add New" → "Project" → выбрать репозиторий `quran-learn` →
   "Deploy".
5. Через 1-2 минуты Vercel выдаст ссылку — сайт готов.

Никаких ключей, баз данных и переменных окружения не требуется — все
данные (текст, перевод, транскрипция, аудио) сайт берёт напрямую с
открытого API alquran.cloud.

## Что дальше

- Добавить остальные суры (меняется один параметр в коде).
- Заменить английскую транскрипцию на кириллическую (по желанию).
- При желании — кнопка "избранное" / отметка выученных аятов (потребует
  базу данных, как в MAST).
